# 🦔 Sonic The Hedgehog 3 - SEGA Mega Drive Web Arcade

¡Bienvenido al repositorio de **Sonic The Hedgehog 3**! Este proyecto es un arcade web minimalista y responsivo que te permite jugar la joya clásica de la SEGA Mega Drive directamente desde tu navegador utilizando la tecnología de **EmulatorJS**.

Desplegado en producción mediante **GitHub Pages**.

---

## 🚀 Características
- **Portabilidad Total:** Juega al instante sin necesidad de instalar emuladores locales o plugins.
- **Optimización Web:** Carga directa gracias al uso de archivos comprimidos (`.zip`) y entrega de núcleos vía CDN estable.
- **Interfaz Limpia:** Diseño oscuro moderno con detalles en azul enfocados al 100% en la experiencia de juego.
